The archivo holds case studies that document the full arc of an analytical project, from the data source through to the findings. Each case study is split into phases so the work can be read in the order it happened, rather than collapsed into a single retrospective. The intent is to make the reasoning visible: not just what the analysis concluded, but where the data came from, what the schema turned out to look like, which queries were run, and which questions ended up worth following further.
